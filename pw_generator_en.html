About the project

This is a maximally simple and minimalist password generator, written in HTML, CSS and JavaScript using ChatGPT and Claude, running locally.

PassQuick password generator is an open source project. You can view its full code on GitHub.

We don't use cookies, don't log and don't transmit user-generated passwords. Everything you do through our generator works locally in your browser.
